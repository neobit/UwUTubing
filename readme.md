## Features
- 💥 Download multiple YouTube links at once
- 🎧 Choose between **MP3** or **MP4**
- ⚡ Fast, simultaneous downloads + conversions
- 📦 Automatic ffmpeg setup (no manual installation!)
- 🐍 Works as Python script or Windows `.exe`

---
